\begin{PROBLEM}
\p
اگر
$a^2, b^2, c^2$
جملات متوالی یک دنباله حسابی باشند، ثابت کنید
$\frac{1}{a + b}, \frac{1}{a + c}, \frac{1}{b + c}$
نیز جملات متوالی یک دنباله حسابی دیگر هستند.
    \SOLUTION{
        \p
       برای سه جمله متوالی یک دنباله حسابی داریم:
        $$a_{n-1} = \frac{a_n + a_{n-2}}{2}$$
        زیرا:
        $$\frac{a_n + a_{n-2}}{2} = \frac{(a_{n - 1} + d) + (a_{n-1} - d)}{2} = a_{n-1}$$
        پس می‌توان نوشت:
        $$a^2 + c^2 = 2b^2$$
        حال باید ثابت کنیم:
        $$\frac{1}{a + b} + \frac{1}{b + c} = \frac{2}{a + c}$$
        $$(2b + a + c)(a + c) = 2(a + b)(b + c)$$
        $$\Downarrow$$
        $$a^2 + c^2 = 2b^2$$
    }
\end{PROBLEM}
