\begin{PROBLEM}
    \p
    فرض کنید 
    $a_{n}$
    تعداد اعداد 
    $n$
    رقمی در مبنای ۳ است؛ اگر تعداد ۰‌ها فرد و تعداد ۱‌ها زوج باشد، 
    $a_{n}$
    را بیابید.

    \SOLUTION[\REF{R} پاسخ]{
        \p
        برای هر عدد (۰ و ۱ و ۲) یک تابع مولد نمایی در نظر می‌گیریم:
        \begin{enumerate}
            \item 
            برای ۰ که باید فرد بار آمده باشد:
                $$G_{1}(x)=\sum\limits_{n=0}^{+\infty} \frac{x^{2n+1}}{(2n+1)!} = \frac{e^{x} - e^{-x}}{2}$$
            \item 
            برای ۱ که باید زوج بار آمده باشند:
                $$G_{2}(x)=\sum\limits_{n=0}^{+\infty} \frac{x^{2n}}{(2n)!} = \frac{e^{x} + e^{-x}}{2}$$
            \item 
            برای ۲ که محدودیتی ندارد:
                $$G_{3}(x)=\sum\limits_{n=0}^{+\infty} \frac{x^{n}}{(n)!} = e^{x}$$
        \end{enumerate}
        \p
        بنابراین، تابع مولد نهایی برابر است با:
        $$G_{1}(x)G_{2}(x)G_{3}(x) = (\frac{e^{x} - e^{-x}}{2})(\frac{e^{x} + e^{-x}}{2})(e^{x}) =$$
        $$\frac{1}{4}(e^{3x} - e^{x}) = \sum\limits_{n=0}^{+\infty} (3^{n} - (-1)^{n}) \frac{x^{n}}{n!}$$	
        \p
        پس پاسخ نهایی برابر خواهد بود با: 
        $$a_{n} = 3^{n} - (-1)^{n}$$
    }
\end{PROBLEM}
