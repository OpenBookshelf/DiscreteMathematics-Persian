\begin{PROBLEM}
    \p
    تعداد جواب‌های معادله‌ی زیر را پیدا کنید.
        $$e_{1} + e_{2} + e_{3} = 17$$
        $$e_{i} \in W, 2 \leq e_{1} \leq 5, 3 \leq e_{2} \leq 6, 4 \leq e_{3} \leq 7 $$

    \SOLUTION{
        \p
        تعداد جواب‌های معادله‌ی بالا با محدودیت‌های در نظر گرفته شده، با ضریب 
        $x^{17}$
        در بسط زیر برابر است:
            $$(x^{2} + x^{3} + x^{4} + x^{5})(x^{3} + x^{4} + x^{5} + x^{6})(x^{4} + x^{5} + x^{6} + x^{7})$$
        زیرا ضریب 
        $x^{17}$
        برابر است با تعداد دفعاتی که حاصل‌ضرب 
        $x^{e_{1}}$
        از عبارت اول و 
        $x^{e_{2}}$
        از عبارت دوم و 
        $x^{e_{3}}$
        از عبارت سوم با
        $x^{17}$
        برابر می‌شود. به عبارت دیگر برابر است با تعداد دفعاتی که 
        $e_{1} + e_{2} + e_{3} = 17$
        می‌شود که با تعداد جواب‌های این معادله برابر است.
        \p
        با ضرب کردن بسط بالا، ضریب 
        $x^{17}$
        برابر ۳ به دست می‌آید. بنابراین معادله دارای ۳ جواب است.
    }
\end{PROBLEM}
