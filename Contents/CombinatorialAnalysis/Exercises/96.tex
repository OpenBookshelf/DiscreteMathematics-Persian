$S_n$
را تعداد افرازهای مجموعه
${1, 2, 3, ..., n}$
به سه مجموعه ناتهی تعریف می کنیم.(برای مثال
$S_4 = 6$
)نشان دهید مقدار
$S_n$
با استفاده از رابطه بازگشتی زیر به دست می‌آید.
(راهنمایی: از قسمت (الف) استفاده کنید.)

$$S_n = 3S_{n-1} + 2^{n-2} - 1, S_1 = 1, S_0 = 0, n > 1$$
